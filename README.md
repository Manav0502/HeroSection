# HeroSection

## Overview

This repository contains a simple Hero Section created using HTML and CSS. The project was built as a beginner-level assignment to practice creating a clean landing page layout. It includes a navigation bar, a heading, a short description, a call-to-action button, and a hero image displayed side by side. The project focuses on using basic HTML elements along with CSS for styling, spacing, colors, and layout without relying on advanced frameworks. It helped me understand how different sections of a webpage are arranged and how CSS can be used to create an attractive and organized interface.

## Files

- `index.html` – Contains the webpage structure.
- `style.css` – Contains all the styling for the webpage.

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.
4. The hero section will be displayed locally without requiring any additional setup.
